What is "ISporta"?

"ISporta" - workout assistant, that helps User and Trainer with workout plans by managing them and seeing workout statistics. Also one of the main purposes is to replace a trainer in real time by system, which will track User workouts anywhere and anytime.

What we use

For pose estimation - "TensorFlow - PoseNet"

Pose estimation is the task of using an ML model to estimate the pose of a person from an image or a video by estimating the spatial locations of key body joints (keypoints).

For front-end programin - "Angular"

Angular is a platform and framework for building single-page client applications using HTML and TypeScript. It implements core and optional functionality as a set of TypeScript libraries that you import into your apps.

For back-end programin - ".NET"

".NET" is a free, cross-platform, open source developer platform for building many different types of applications.
